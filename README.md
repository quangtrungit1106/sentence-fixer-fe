# Cách setup để chạy project

## 1. Cài đặt dependencies:

```bash
$ npm install
```
## 2. Tạo file .env với nội dung:

```ini
VITE_API_BASE_URL=http://localhost:8080/
```
## 3. Chạy ứng dụng:

```bash
$ npm run dev
```
